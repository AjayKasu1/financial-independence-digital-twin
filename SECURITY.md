# Security policy

Do not open a public issue for a suspected vulnerability or include client data, credentials, tokens, or infrastructure identifiers in a report. Contact the repository owner privately with a minimal reproduction, affected component, impact, and suggested mitigation.

This repository is a synthetic-data demonstration and has not been certified for real client records. Before production use, complete the controls in [`docs/threat-model.md`](docs/threat-model.md) and [`docs/data-governance.md`](docs/data-governance.md).
